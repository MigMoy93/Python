# 🐍 Snake Game

A classic Snake game built in Python using the `turtle` module.  
Made as a learning project while studying DAM (Software Development).

![gameplay](assets/LobbyScreen.png)

---

## 🚀 How to run

```bash
python main.py
```

> Requires Python 3.x — no external libraries needed.  
> ⚠️ Sound effects only work on Windows (`winsound`).

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| Arrow keys | Move the snake |
| Space | Pause / Resume |
| 1 / 2 / 3 | Difficulty (in menu) |
| C | Change color (in menu) |
| F | Change shape (in menu) |

---

## 🗂️ Project structure

| File | Description |
|------|-------------|
| `main.py` | Game loop and input handling |
| `snake.py` | Snake class |
| `food.py` | Food class |
| `scoreboard.py` | Score display |
| `menu.py` | Start menu |
| `sound.py` | Sound effects |

---

## 🛠️ Built with

- Python 3
- `turtle` — graphics and movement
- OOP — each element is its own class
